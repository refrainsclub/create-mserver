# create-mserver

This project helps you create a Paper server.

It automates mundane tasks such as downloading the server, accepting
the Mojang EULA, and creating startup scripts with Aikar's Flags.

## Planned Features

- Installing library plugins
- Disabling the nether and end dimension
- Server configuration templates (minigame template, survival template, etc)
